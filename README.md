📊 Attendance Management System

This project is a Django-based web application designed to manage and track student attendance efficiently.

🚀 Features:

* Add and manage students
* Mark attendance (Present/Absent)
* Store attendance data in database
* View attendance records
* Clean UI for easy usage

🛠️ Tech Stack:

* Python
* Django
* HTML, CSS
* SQLite

📌 Purpose:
To understand how to manage and store real-time data like attendance using Django.

📂 Use Case:
This system can be used in schools, colleges, or organizations to track attendance records digitally.
